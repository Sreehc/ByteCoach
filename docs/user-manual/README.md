# 用户指南

> OfferPilot 使用指南，帮助你从训练、简历到投递完整使用当前系统能力。

## 快速开始

1. 注册账号并登录。
2. 进入首页工作台，先看系统建议你当前最该处理的动作。
3. 从题库、知识库、问答或模拟面试开始一轮训练。
4. 用学习计划安排后续节奏，再用简历助手和投递管理把训练结果转成真实求职动作。

如果你是第一次使用，优先顺序建议是：题库或知识库 -> 问答或面试 -> 学习计划 -> 简历助手 -> 投递管理。

## 主线功能

### 首页工作台

进入「首页」后，先确认两件事：

- 当前最应该先做什么
- 哪些训练、简历或投递事项还没推进

你可以在这里快速进入：

- 题库训练
- 知识库和问答
- 模拟面试
- 学习计划
- 简历助手
- 投递管理

如果首页提示你有待复习、待执行计划或待推进投递，建议先处理这些动作，再开始新的训练。

### 题库训练

进入「题库」页面后，你可以：

- 按关键字、分类、题型、难度、岗位方向和标签筛题
- 打开题目详情，直接看标准答案、面试表达、追问建议和常见错误
- 快速判断某个知识点是否已经掌握

如果你还不确定先练什么，先筛出一个方向，再看高频题和常见错误。

### 知识库

进入「知识库」页面后，你可以：

- 浏览平台资料
- 上传自己的 Markdown、TXT、PDF、Word 文档
- 查看文档当前状态，确认是否已经可用于问答或后续生成卡片
- 删除自己上传的资料

适合上传的内容包括：

- 面试笔记
- 项目资料
- 简历补充材料
- 零散知识文档

如果你希望问答更贴近自己的资料，先在这里上传文档，再去「问答」页面继续提问。

### 问答

进入「问答」页面后，你可以选择不同提问路径：

- 直接提问：适合快速确认某个知识点
- 基于资料提问：适合让回答引用知识库内容
- 基于简历或项目提问：适合围绕你的经历继续追问

建议做法：

- 想快速补概念，直接问
- 想让回答更贴近你的资料，先选资料范围
- 想练项目表达，优先绑定简历或项目上下文

### 模拟面试

进入「模拟面试」页面后，你可以：

- 选择文字模式或语音模式
- 选择方向、题数和上下文
- 直接开始作答
- 在结束后查看评分、点评、标准答案和追问

适合的使用方式：

- 刷完一轮题库后，用面试验证表达
- 上传简历后，用项目相关问题检查讲述是否完整
- 完成一轮面试后，去「面试历史」或「面试详情」做复盘

### 学习计划

进入「学习计划」页面后，你可以：

- 生成 7 / 14 / 30 天计划
- 查看今天要做的任务和预估时长
- 标记任务完成
- 在训练重点变化后刷新计划

如果你还没有明确节奏，先生成一份计划；如果你已经有计划，优先完成今天的任务。

### 简历助手

进入「简历助手」页面后，你可以按顺序完成：

1. 上传 PDF / DOC / DOCX 简历
2. 检查解析结果是否完整
3. 查看项目拆分和追问建议
4. 生成或修改自我介绍
5. 查看面试简历提纲

如果你有多份简历，可以切换历史记录，对比不同版本的解析结果。

### 投递管理

进入「投递管理」页面后，你可以：

- 记录新的岗位投递
- 填写公司、岗位、城市、渠道和 JD
- 查看当前推进中的岗位
- 更新投递状态
- 记录笔试、面试、复盘和备注事件

点击单条投递后，建议优先看两块内容：

- 当前状态和下一步要推进什么
- 时间线里已经发生了哪些事件

### 数据分析

进入「数据分析」页面后，你可以：

- 回看训练趋势
- 查看计划执行、简历准备和投递进展相关洞察
- 判断最近的训练节奏是否稳定

它更适合阶段复盘，不是主入口。建议在完成几轮训练或推进几条投递后再集中查看。

## 辅助功能

### 卡片强化

进入「卡片强化」页面后，你可以：

- 查看当前激活 deck
- 处理今日卡片任务
- 对卡片进行评分
- 切换或激活新的 deck

适合在主线训练之外补强记忆，不替代题库、问答或面试。

### 复习中心

进入「复习中心」页面后，你可以：

- 查看今天待复习的内容
- 按 `1=重来 / 2=困难 / 3=良好 / 4=简单` 提交评分
- 查看当前复习统计

如果今天待复习内容较多，建议先清掉高优先级内容，再开始新的训练。

### 错题本

错题主要来自面试或训练过程中的低分题。你可以：

- 回看错题详情
- 更新掌握状态
- 导出为 Markdown 继续整理

### 社区

进入「社区」页面后，你可以：

- 浏览问题和回答
- 发起提问
- 对问题和回答投票
- 查看排行榜

社区用于外部交流，不替代你在主线中的训练、简历和投递动作。

## 账户与设置

进入「设置」页面后，可以管理账号和安全能力。

### 账户资料

- 查看当前用户信息
- 上传头像
- 验证邮箱

### 设备管理

- 查看当前登录设备
- 撤销指定设备
- 撤销其它所有设备

### 登录历史

- 查看登录时间、IP、城市、成功失败状态

### 两步验证

- 初始化并启用 2FA
- 使用验证码或恢复码完成登录校验
- 需要时关闭 2FA

### 数据导出

- 导出个人训练数据

## 管理后台

> 仅管理员可见。

进入「管理后台」后，你可以直接执行治理动作：

- 系统概览：看整体趋势和负载
- 用户管理：查看、编辑、封禁、解封
- 内容审核：处理待审核内容
- 分类管理：维护题库和知识库分类
- 题库管理：维护题目与批量导入
- 文档管理：导入资料、重切分、重建索引
- AI 日志：查看模型调用日志和摘要
- 系统配置：维护模型与提示词覆盖项
- 面试治理：查看面试会话质量和摘要
- 登录日志：查看全局登录日志

## 常见问题

**Q: 忘记密码怎么办？**  
A: 在登录页进入「找回密码」，通过注册邮箱接收验证码后重置密码。

**Q: 为什么知识库问答没有引用我的资料？**  
A: 先确认文档已经上传成功，并且解析与索引已完成。你也可以在知识库页面查看自己的文档列表和状态。

**Q: 语音面试无法使用怎么办？**  
A: 先检查浏览器麦克风权限，再确认语音能力状态是否可用。

**Q: 简历上传支持哪些格式？**  
A: 当前支持 `PDF / DOC / DOCX`。

**Q: 投递状态可以修改吗？**  
A: 可以，在投递详情中更新状态，并继续补充事件记录。
