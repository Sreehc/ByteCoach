# 用户指南

> ByteCoach 使用指南，帮助你快速上手所有功能。

## 快速开始

1. 注册账号并登录
2. 首页 Dashboard 会展示学习概览，首次访问会有引导卡片
3. 从左侧导航栏选择功能模块开始学习

## 功能导航

### 智能问答

进入「问答」页面，支持两种模式：

- **普通问答**：直接向 AI 提问 Java 面试相关问题
- **知识库问答（RAG）**：基于内置知识库回答，答案会附带引用来源

可以在输入框切换模式。知识库问答的回答更准确，因为它是基于上传的学习资料生成的。

### 模拟面试

进入「面试」页面，选择面试方向和题数（3-5 题）：

- **文字面试**：输入文字回答，AI 实时评分、点评和追问
- **语音面试**：点击录音按钮语音作答，系统自动转写为文字

每道题评分后会展示标准答案和点评，低于 60 分的题目自动加入错题本。

面试完成后可在「面试历史」查看所有记录和复盘。

### 错题本

进入「错题本」页面，查看所有面试中答错的题目：

- 每道错题显示掌握程度（未开始 / 复习中 / 已掌握）
- 可以直接查看标准答案和错误原因
- 错题会自动进入间隔复习调度

### 间隔复习

进入「复习」页面，系统基于 SM-2 遗忘曲线算法自动安排复习：

- 每天展示需要复习的题目
- 复习后评分（1=重来 2=困难 3=良好 4=简单）
- 系统根据评分自动调整下次复习间隔
- 连续答对间隔会越来越长，真正快忘记时才复习

### 学习计划

进入「学习计划」页面：

- 点击「生成计划」，AI 会根据你的薄弱点和错题自动生成多日学习计划
- 每天有具体的任务（面试、复习、知识点学习）
- 完成任务后标记为已完成
- 如果连续完成率低，系统会自动调整计划

### 知识库

进入「知识库」页面，浏览和搜索内置学习资料：

- 支持按分类筛选（Spring、JVM、MySQL 等）
- 搜索支持关键词匹配
- 管理员可以上传新的学习资料

### 学习社区

进入「社区」页面：

- **浏览问题**：查看社区中的面试问题和讨论
- **发起提问**：点击「发起提问」发布你的问题
- **回答问题**：在问题详情页提交回答
- **投票**：对有价值的问题和回答点赞
- **排行榜**：查看社区贡献排名和等级

社区积分规则：提问 +2、回答 +5、被采纳 +20、被点赞 +1。

### 数据分析

进入「数据分析」页面，深入了解你的学习模式：

- **趋势分析**：按周查看得分趋势和面试频次
- **效率分析**：SM-2 遗忘率、EF 趋势、掌握度分布
- **时段分析**：不同时间段的学习效果对比

## 账户设置

进入「设置」页面（侧边栏底部或点击头像），管理你的账户安全：

### 设备管理

- 查看当前已登录的所有设备（设备名、IP、城市、最后活跃时间）
- 当前设备标记为「当前设备」
- 可以撤销单个设备或撤销除当前设备外的所有设备

### 登录历史

- 查看所有登录记录（时间、IP、城市、设备、成功/失败状态）

### 两步验证

- 启用两步验证后，登录时需要输入身份验证器应用中的 6 位验证码
- 支持 Google Authenticator、Microsoft Authenticator 等
- 启用时会生成 8 个恢复码，请妥善保存
- 可以随时关闭两步验证（需要验证当前验证码）

### 数据导出

- 导出你的学习数据为 Excel 文件
- 包含 3 个 Sheet：面试记录、错题本、复习记录

## 管理后台

> 仅管理员账号可见。入口在侧边栏底部「管理后台」。

### 系统概览

- 总用户数、今日活跃、今日新增、总面试次数、总复习次数
- 近 30 天用户增长和活跃趋势折线图

### 用户管理

- 搜索用户（按用户名/昵称）
- 按角色筛选（管理员/普通用户）
- 编辑用户信息（昵称、角色）
- 封禁/解封用户（封禁后踢出所有设备）
- 查看用户详情（注册时间、最后登录、学习统计）

### 内容审核

- 社区内容发布后默认为「待审核」状态
- 审核通过后内容对外可见
- 审核拒绝需填写原因，自动通知作者

### 分类管理

- 管理题库/知识库/面试的分类体系

### 题库管理

- 题目的增删改查
- 按分类和难度筛选
- **批量导入**：上传 Excel 文件批量导入题目

### 文档管理

- 内置知识资料的管理
- 支持导入、重新切分、重建索引

### 登录日志

- 查看所有用户的登录记录
- 支持按用户名搜索

### 数据导出

- 导出题库为 Excel
- 导出用户列表为 Excel

## 常见问题

**Q: 忘记密码怎么办？**
A: 目前暂不支持自助重置密码，请联系管理员重置。

**Q: 两步验证的恢复码用完了怎么办？**
A: 登录后进入设置 → 两步验证，关闭后重新启用即可生成新的恢复码。

**Q: 面试评分不准确怎么办？**
A: AI 评分基于 LLM，可能存在偏差。评分仅供参考，建议结合标准答案自行判断。

**Q: 知识库问答找不到相关内容？**
A: 知识库问答基于已上传的文档，如果相关知识未被收录则无法回答。可以联系管理员上传新的学习资料。
