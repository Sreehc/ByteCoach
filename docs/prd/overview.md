# ByteCoach 产品总览

## 项目定位

ByteCoach 是一个面向 Java 后端工程师的面试准备平台，通过 AI 驱动的问答、模拟面试、错题复习、学习计划四段闭环，帮助用户系统性地备战技术面试。

## 目标用户

- 准备 Java 后端面试的工程师（初级/中级/高级）
- 需要系统性复习而非碎片化刷题的学习者

## 核心价值

- **闭环学习**：问答 → 面试 → 错题 → 复习 → 计划，每个环节的数据自动流入下一环节
- **AI 驱动**：面试评分、追问、计划生成、自适应推荐均基于 LLM
- **间隔复习**：基于 SM-2 算法自动调度复习时机，避免遗忘曲线

## 功能清单

### 一、学习核心

| 模块 | 功能 | 说明 |
|------|------|------|
| 智能问答 | 普通问答 | 基于 LLM 的 Java 面试知识问答 |
| 智能问答 | 知识库问答（RAG） | 基于上传文档的增强问答，支持引用溯源 |
| 模拟面试 | 文字面试 | 3-5 题/场，AI 实时评分、点评、追问 |
| 模拟面试 | 语音面试 | Whisper STT 转写 + TTS 播放，支持语音作答 |
| 模拟面试 | 面试历史 | 查看所有已完成面试的详情和复盘 |
| 错题本 | 自动归档 | 面试得分低于阈值自动加入错题本 |
| 错题本 | 掌握程度追踪 | not_started → reviewing → mastered 三阶段 |
| 间隔复习 | SM-2 调度 | 基于遗忘曲线自动计算下次复习日期 |
| 间隔复习 | 复习评分 | 1-4 分评价，动态调整 EF 和间隔 |
| 学习计划 | AI 生成 | 根据薄弱点和错题自动生成多日学习计划 |
| 学习计划 | 自动调整 | 连续完成率低时触发计划重新生成 |
| 学习计划 | 版本追踪 | 记录计划调整历史 |

### 二、知识管理

| 模块 | 功能 | 说明 |
|------|------|------|
| 知识库 | 文档管理 | 上传/导入学习资料（PDF/TXT/MD） |
| 知识库 | 自动切分 | 文档按 token 数自动分片 |
| 知识库 | 向量索引 | Embedding 向量化 + Redis 向量搜索 |
| 知识库 | 内置资料 | Spring 核心、JVM 面试、MySQL 高频题等预置内容 |
| 题库 | CRUD | 题目的增删改查，按分类和难度管理 |
| 分类管理 | 三级分类 | question / knowledge / interview 三种类型 |

### 三、社区互动

| 模块 | 功能 | 说明 |
|------|------|------|
| 社区 | 发起提问 | 发布面试相关问题 |
| 社区 | 回答问题 | 回答他人提问，支持采纳 |
| 社区 | 投票 | 对问题和回答点赞 |
| 社区 | 排行榜 | 按社区贡献积分排名 |
| 社区 | 等级体系 | 见习生 → 资深工程师，基于学习和社区行为 |

| 模块 | 功能 | 说明 |
|------|------|------|
| Dashboard | 学习概览 | 面试次数、平均分、错题数、计划完成率 |
| Dashboard | 薄弱点分析 | 按分类展示能力得分和薄弱项 |
| Dashboard | 最近面试 | 最近面试记录快速入口 |
| 数据分析 | 趋势分析 | 按周统计得分趋势和面试频次 |
| 数据分析 | 效率分析 | SM-2 EF 趋势、遗忘率、掌握度分布 |
| 数据分析 | 时段分析 | 不同时间段的学习效果对比 |
| 自适应推荐 | 能力画像 | 按分类计算能力得分 |
| 自适应推荐 | 智能推荐 | 推荐适合当前水平的面试方向和题目 |

## 迭代历史

### 一期：核心学习闭环

建立项目基础架构和核心学习功能。

- 技术栈搭建：Spring Boot 3 + Vue 3 + MyBatis-Plus + Element Plus
- 用户注册/登录（JWT）
- 智能问答（普通问答 + RAG 知识库问答）
- 模拟面试（文字模式，AI 评分/追问）
- 错题本（自动归档 + 掌握程度追踪）
- 间隔复习（SM-2 算法调度）
- 学习计划（AI 生成 + 自动调整）
- Dashboard 学习概览

### 二期：社区与知识库

扩展社区互动和知识管理能力。

- 知识库系统（文档上传、切分、向量索引、内置资料）
- 学习社区（提问、回答、投票、排行榜、等级体系）
- 题库管理（CRUD、分类体系）
- 管理后台（分类管理、题库管理、文档管理）
- 通知系统（站内通知、定时提醒）

### 三期：进阶学习功能

增强学习体验和个性化推荐。

- 语音面试（Whisper STT + TTS）
- 数据分析（趋势、效率、时段分析）
- 自适应推荐（能力画像、智能推荐）
- 面试历史与复盘

### 四期：生产就绪

补齐安全短板、强化管理后台、实现容器化部署。

- 登录设备管理（设备列表、远程撤销）
- 登录异常检测（IP 地理位置、异地通知、失败锁定、登录日志）
- 两步验证（TOTP + 恢复码）
- 用户管理（列表、编辑、封禁/解封、详情）
- 内容审核（待审核列表、通过/拒绝）
- 系统概览（统计卡片、趋势图）
- Docker 化（多阶段构建、docker-compose 编排）
- 数据管理（Excel 导出/导入）
- 管理后台权限控制（仅管理员可见）
