# ByteCoach 技术文档

本目录包含 ByteCoach 项目的核心技术设计文档。

## 文档目录

1. [系统架构设计](./03-系统架构设计.md) — 整体架构、分层设计、技术选型
2. [数据库设计](./04-数据库设计.md) — 表结构、索引约束、ER 关系
3. [接口设计](./05-接口设计.md) — RESTful API 定义、请求响应格式

## 阅读建议

- 先读「系统架构设计」，了解整体分层和模块职责
- 再读「数据库设计」，理解数据模型和表关系
- 最后读「接口设计」，明确前后端交互契约

完整 API 文档可通过后端 Swagger 获取：http://localhost:8080/doc.html
