# ByteCoach 文档总览

- 第一步：先读“项目规划与功能评审”，明确项目定位与闭环。

## 当前版本目标

- 项目名称：ByteCoach
- 阶段定位：一期 MVP 文档基线
- 目标场景：求职作品集 / 项目答辩 / Java 智能学习 Agent 原型
- 实施原则：
  - 先跑通业务闭环，再增强 AI 深度
  - 先保证演示效果，再扩展复杂工程能力
  - 先用内置知识库，再开放用户自建知识库
