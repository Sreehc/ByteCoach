# 前端改版实施进度

## 模块 1：核心学习主线收口

状态：已完成

### 本轮目标

- 收紧 `Dashboard / Cards / Review / Knowledge` 的首屏信息
- 删除重复说明，保留单一主动作
- 统一 `deck` 为中文表达
- 移除普通用户路径中的调试型内容

### 已完成调整

#### 全局页头

- `首页 / 知识库 / 今日卡片 / 复习中心` 去掉了页面级副标题
- `Dashboard` 页面标题恢复为更中性的 `首页`，主任务交给内容区 Hero 承担

#### Dashboard

- 首次使用引导卡压缩为单标题
- 无卡片任务时不再和主 Hero 重复提示
- 快捷入口卡从 `标签 + badge + 标题 + 描述` 压缩为 `标签 + 标题`

#### Cards

- 无卡组时只保留一个主空状态，不再展示多块空面板
- `deck` 全部改为 `卡组`
- 生成卡片后的动作统一为 `设为当前卡组`
- 无任务状态下改为明确引导去知识库

#### Review

- 删除页头重复的 `开始复习` 按钮
- 收短无数据标题和说明文案

#### Knowledge

- 空状态说明缩短
- 文档卡中的 `还没有生成卡片` 改为 `未生成卡片`
- `进入今日卡片` 改为 `去今日卡片`
- `检索测试` 仅对管理员可见

### 本轮结果

- 核心学习链路首屏更聚焦
- 无数据状态下的用户路径更直接
- 文案重复明显下降
- 普通用户视角下的知识库边界更干净

## 模块 2：学习辅助页收口

状态：已完成

- 收紧 `Chat / Interview / Analytics / Community` 的页头和分区标题

- 让空状态、按钮和输入提示改成更短的动作型表达

### 已完成调整

#### 全局页头

- `问答 / 面试诊断 / 数据分析 / 学习社区` 去掉了页头副标题
- `面试历史 / 面试详情 / 发起提问 / 问题详情 / 社区排行榜` 也同步去掉了副标题

#### Chat

- 会话空状态和首屏空状态改成更短文案
- 输入区模式提示收短
- 引用抽屉去掉重复解释

#### Interview

- 设置区只保留标题，不再叠加说明段
- 开始前占位页压缩为单标题
- 评分中、结果页、完成页去掉重复分区标题
- 回答区输入提示收短

#### Analytics

- 主图区和各分析分区统一改成单标题
- 删除大段解释型说明
- 顶部摘要卡不再附带重复解释文案

#### Community

- 社区列表搜索、空状态和回复区文案收短
- 提问页侧边说明从三条列表压缩为单句提示
- 排行榜页去掉“榜首摘要”说明段

### 本轮结果

- 学习辅助页的视觉层级更稳定
- 页头不再和内容区重复解释同一件事
- 空状态和输入区更像动作入口，而不是说明书
- 社区与分析页的信息密度更可控

### 下一模块

- `Settings / Admin`
- 重点处理管理型页面的布局密度、状态反馈和文案一致性
