# 前端改版实施进度

## 模块 1：核心学习主线收口

状态：已完成

### 本轮目标

- 收紧 `Dashboard / Cards / Review / Knowledge` 的首屏信息
- 删除重复说明，保留单一主动作
- 统一 `deck` 为中文表达
- 移除普通用户路径中的调试型内容

### 已完成调整

#### 全局页头

- `首页 / 知识库 / 今日卡片 / 复习中心` 去掉了页面级副标题
- `Dashboard` 页面标题恢复为更中性的 `首页`，主任务交给内容区 Hero 承担

#### Dashboard

- 首次使用引导卡压缩为单标题
- 无卡片任务时不再和主 Hero 重复提示
- 快捷入口卡从 `标签 + badge + 标题 + 描述` 压缩为 `标签 + 标题`

#### Cards

- 无卡组时只保留一个主空状态，不再展示多块空面板
- `deck` 全部改为 `卡组`
- 生成卡片后的动作统一为 `设为当前卡组`
- 无任务状态下改为明确引导去知识库

#### Review

- 删除页头重复的 `开始复习` 按钮
- 收短无数据标题和说明文案

#### Knowledge

- 空状态说明缩短
- 文档卡中的 `还没有生成卡片` 改为 `未生成卡片`
- `进入今日卡片` 改为 `去今日卡片`
- `检索测试` 仅对管理员可见

### 本轮结果

- 核心学习链路首屏更聚焦
- 无数据状态下的用户路径更直接
- 文案重复明显下降
- 普通用户视角下的知识库边界更干净
